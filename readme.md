### Тестовое задание в SAKH.COM на должность junior php-разработчик.
Что бы запустить мой проект: 
1) Склонируйте репозиторий, 
2) Переименуйте файл .env.example в .env и настройте соединение с базой данных
3) Выполните команду ```php artisan migrate```
4) Выполните команду ```php artisan db:seed``` 

(Log файл создается в папке /storage/logs/)
